# Business Requirements

## Overview

The platform is an enterprise-grade multi-tenant SaaS solution for gyms, fitness centers, trainers, dietitians, and individual users.

---

# User Types

- Super Admin
- Admin (Gym Owner)
- Trainer
- Dietitian
- Client
- Individual User

---

# Core Features

## Authentication

- Login
- Register
- Forgot Password
- Reset Password
- Email Verification
- JWT Authentication
- Refresh Token

---

## Multi-Tenant

- One platform for multiple gyms
- Complete tenant isolation
- Separate dashboards for each tenant

---

## Dashboard Management

Admin can create dashboards based on subscription.

### Basic Plan

- 1 Dashboard

### Professional Plan

- 2 Dashboards

### Enterprise Plan

- Unlimited Dashboards

Each dashboard has separate:

- Clients
- Trainers
- Dietitians
- Reports
- Analytics
